Quick Help
  1. Each line in the text is for one subtitle caption. It will show on the screen as one caption. Both text-to-speech and automatic translation sees each line as a new sentence. So, each line must be a complete phrase for translation.
  2. By default, there will be pauses between sentences, matching natural speech. You can add an extra pause by inserting a line that looks like #N, where N is the pause length in milliseconds. For example, #2000 means a pause of 2000 milliseconds or 2 seconds. This is useful if there is an action happening between sentences in the video, like a slide change. The "#" symbol must be the first character in the line, and the line should not have any other text except the pause time.
  3. You can set the exact time from the start of the sound file for each line. Use the format #HH:MM:SS,SSS to show when the phrase begins, with millisecond accuracy. For example, #00:01:35.5 means the phrase on the next line starts at 1 minute and 35.5 seconds after the beginning. Make sure all the sentences are in the correct order.
  4. You can use Speech Synthesis Markup Language (SSML) to make sure that abbreviations or words in foreign languages pronounced correctly.
    See SSML guide and examples

    Insert two seconds pause inside the phrase: Hello there.<break time="2s"/> How are you today?
    Pronounce "ICP" as abbreviation: <say-as interpret-as="characters">ICP</say-as> is the abbreviation for Infineon Cloud Platform.
    Say in other language: I can speak French. <lang xml:lang="fr-FR">Je ne parle pas français.</lang>
    Use phonetic alphabet: You say, <phoneme alphabet="ipa" ph="pɪˈkɑːn">pecan</phoneme>.
    Some special cases: 2 devided by 9 is pronounced <say-as interpret-as="fraction">2/9</say-as>.

    For full SSML reference, visit AWS Polly Supported SSML Tags

  5. Place the cursor on any line and press the "Play current line" button, or use ALT-P to hear how the single phrase sounds.
